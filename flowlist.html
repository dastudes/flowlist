Welcome to FlowList

Your lists sync across devices via a JSON file in your GitHub repo. One-time setup takes about two minutes.

1
Create a repo called flowlist at github.com/new. Make it public and check "Add a README file."
2
Go to GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic). Create a token with the repo scope checked.
3
Paste your token below. It's stored only in this browser's local storage.
