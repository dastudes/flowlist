Welcome to FlowList

Your lists sync across devices via a JSON file in your GitHub repo. One-time setup takes about two minutes.

1
Create a new GitHub repo called flowlist at github.com/new. Make it public. Check "Add a README file."
2
Go to GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic). Create a token with repo scope. Copy it.
3
Paste it below. Your token is stored only in this browser's local storage — never sent anywhere except GitHub's API.
